# EuTxGNN - EU Drug Repurposing Predictions

AI-powered drug repurposing predictions for European Medicines Agency (EMA) approved drugs using TxGNN.

{: .warning }
> **Research Use Only**: These predictions are for research purposes only and do not constitute medical advice. All candidates require clinical validation before any therapeutic application.

---

## Overview

EuTxGNN leverages the [TxGNN](https://www.nature.com/articles/s41591-023-02233-1) knowledge graph and deep learning model to identify potential new therapeutic uses for existing EMA-approved medications.

### Key Statistics

| Metric | Value |
|--------|-------|
| Total Predictions | 32,368 |
| Unique Drugs | 638 |
| Unique Indications | 4,570 |
| KG Predictions | 718 |
| DL Predictions | 31,650 |

---

## Data Sources

- **EMA Medicines Database**: Centrally authorized human medicines
- **Article 57 Database**: EU pharmaceutical product submissions
- **TxGNN Knowledge Graph**: Drug-disease relationships from multiple biomedical databases
- **DrugBank**: Drug target and interaction data

## Methodology

### Knowledge Graph (KG) Predictions

Identifies drug-disease associations based on network topology and known relationships in the biomedical knowledge graph.

### Deep Learning (DL) Predictions

Uses graph neural networks trained on the TxGNN model to predict novel drug-disease associations with confidence scores.

### Evidence Levels

| Level | Description |
|-------|-------------|
| L1 | FDA/EMA approved indication |
| L2 | Phase III clinical trial |
| L3 | Phase I/II clinical trial |
| L4 | Published case reports |
| L5 | Computational prediction (TxGNN) |
